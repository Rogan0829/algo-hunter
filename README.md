# 🎯 Algo Hunter

**스마트스토어 상품명 알고리즘 최적화 분석기**

네이버 스마트스토어 셀러를 위한 무료 상품명 분석 도구입니다.  
상품명을 입력하면 알고리즘 최적화 점수와 구체적인 개선 제안을 즉시 받을 수 있습니다.

## 🔍 분석 항목

- **상품명 길이** — 최적 길이(25-35자) 여부
- **금칙어 검사** — 노출 제한 유발 단어 감지
- **특수문자** — 스팸 필터링 위험 감지
- **키워드 최적화** — 타겟 키워드 위치/포함 여부
- **가독성** — 중복 단어, 구조 분석

## 🚀 바로 쓰기

```bash
pip install streamlit
streamlit run app.py
```

## 📊 점수 기준

| 등급 | 점수 | 의미 |
|------|------|------|
| S | 85-100 | 최상위 노출 최적화 |
| A | 70-84 | 양호, 소폭 개선 가능 |
| B | 55-69 | 보통, 개선 필요 |
| C | 40-54 | 미흡, 즉시 수정 권장 |
| D | 0-39 | 위험, 노출 심각하게 낮음 |
